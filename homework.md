个人对继承的理解：这个项目因为实现也是自己写，所以继承最大的作用体现不出来。继承最大作用是偷懒 你只需要提供一个接口，比如
```
class course
{
 public:
    // 写成纯虚函数，类似java接口，这样所有的函数都可以让调用人来写了
    virtual void PrintHelpInfo() = 0;
}
```
